# בריף עיצוב — ספריית מדיה פנימית (DAM)

> **מטרה:** עיצוב ספריית מדיה מרכזית + Media Picker אוניברסלי לשימוש חוזר.
> **תפקיד:** עמוד חדש בניהול התוכן — instructor+.
> **מקורות:** SRS §1 (MediaAsset), UploadFile/CreateFileSignedUrl · אחסון: Cloudflare R2 (קיים).

---

## 1. הניתוח בקצרה — זו שכבת DAM, לא רק עמוד

`MediaAsset` קיים (`title, file_url, file_type, file_size, tags`) אך מדיה מפוזרת
גם כ-URLs גולמיים (`image_url`, `media[]`, ב-blocks/שאלות/צמתים). **הספרייה
שווה משהו רק אם כל העלאה עוברת דרכה.** שני חלקים:

1. **עמוד הספרייה** — הגלריה הגלויה.
2. **Media Picker אוניברסלי** — רכיב שכל נקודת "הוסף מדיה" פותחת (הפרס האמיתי).

**הרחבה מומלצת ל-MediaAsset:** `topic/category` (למיון לפי נושאים), `dimensions`,
`thumbnail_url`, `alt`, ומעקב שימוש (`usage[]`/`usage_count`).

**דפוס אובייקט-משותף (3):** מחיקת נכס בשימוש → אזהרה (כמו SharedModule, Question).

---

## 2. עמוד הספרייה — מבנה

- כותרת "ספריית מדיה" + כפתור "העלה מדיה" (drag-drop, bulk).
- חיפוש + סינון: סוג (תמונה/GIF/וידאו/PDF), תגיות, נושא, תאריך, בשימוש/לא-בשימוש.
- מיון: לפי נושא / חדש / הכי בשימוש.
- גלריית thumbnails: כל נכס עם תגית-סוג, וכותרת. hover → פעולות מהירות.
- בחירת נכס → פאנל פרטים: תצוגה מקדימה גדולה, מטא (שם, סוג, גודל, מימדים,
  תאריך, מעלה), תגיות ונושא (לעריכה), "היכן בשימוש" (רשימת הפניות),
  העתק URL, הורד, החלף, מחק (עם אזהרת שימוש).
- פעולות bulk: תייג מרובים, מחק מרובים.

---

## 3. Media Picker — מבנה (מודאל)

נפתח מכל נקודת הוספת מדיה במערכת:
- אותה גלריה + סינון, במצב **בחירה**.
- לשונית "העלה חדש" → מעלה → רושם MediaAsset → מחזיר.
- בחירה (יחיד/מרובה) → מחזיר `file_url` להקשר הקורא.

## 📋 פרומפט A — עמוד הספרייה (העתק מכאן)

```
אני רוצה לעצב את ספריית המדיה הפנימית של iPracticom Academy.
השתמש בשפת העיצוב והטוקנים הקיימים — עקביות מלאה, RTL.

— מותג: עברית RTL · #2EB4FF / #0075DB / #757D86 / #181D24 / רקע #F4FBFF ·
  tech-human · אייקוני outline · אנימציות עדינות.

— מבנה —
• כותרת "ספריית מדיה" + כפתור "העלה מדיה" (drag-drop, ריבוי קבצים).
• שורת חיפוש + פילטרים: סוג (תמונה/GIF/וידאו/PDF), תגיות, נושא, בשימוש/לא-בשימוש.
• בורר מיון: לפי נושא / נוסף לאחרונה / הכי בשימוש.
• גלריית thumbnails ברשת: כל נכס עם תגית-סוג בפינה, כותרת, ובמעבר עכבר —
  פעולות מהירות (העתק URL, הורד, מחק). וידאו עם אייקון play; PDF עם אייקון מסמך.
• פאנל פרטים (בבחירת נכס): תצוגה מקדימה גדולה, מטא (שם, סוג, גודל, מימדים,
  תאריך העלאה, מעלה), עריכת תגיות ונושא, רשימת "היכן בשימוש" (הפניות),
  וכפתורי העתק URL / הורד / החלף / מחק.
• אזהרה במחיקת נכס בשימוש: "נכס זה בשימוש ב-N מקומות".

— נתוני דמו (עברית) —
תמונות: "מצלמת אבטחה - חיבור PoE", "לוח MikroTik", "תרשים רשת".
GIF: "הבהוב נורית LED". וידאו: "התקנת מצלמה - הדרכה". PDF: "מדריך התקנה".
תגיות: רשתות, מצלמות, MikroTik, התקנה.

— מצבים: דסקטופ-first + מובייל. הצג עם פאנל פרטים פתוח על תמונה.
— טכני: React + Tailwind + shadcn/ui, lucide-react (outline), framer-motion.
  שמור על רכיבים קיימים.

תתחיל מהגלריה המאוכלסת בדסקטופ עם פאנל פרטים פתוח.
```

## 📋 פרומפט B — Media Picker (אחרי שהספרייה מוכנה)

```
עכשיו עצב את ה-Media Picker — מודאל שנפתח מכל מקום במערכת שמוסיפים בו מדיה
(בלוק בעורך שיעור, צומת ב-Playbook, מדיה בשאלה, תמונת פרופיל, לוגו תעודה).
אותה שפת עיצוב, RTL.

— מבנה —
• מודאל עם שתי לשוניות: "בחר מהספרייה" ו-"העלה חדש".
• "בחר מהספרייה": אותה גלריה + סינון (סוג/תגיות/נושא/חיפוש), במצב בחירה.
  כל נכס לחיץ; תמיכה בבחירה יחידה ומרובה (לפי ההקשר).
• "העלה חדש": אזור drag-drop → תצוגת התקדמות → שדות תגיות/נושא → "הוסף".
• כפתור אישור מחזיר את הנכס/ים שנבחרו.

— הצג את המודאל פתוח על לשונית הספרייה עם נכס אחד מסומן.
— טכני: React + Tailwind + shadcn. שמור על רכיבים קיימים.
```

## אנטומיה — רפרנס שדות

| אלמנט | שדה (קיים / *מורחב) |
|---|---|
| כותרת | `MediaAsset.title` |
| URL | `MediaAsset.file_url` |
| סוג | `MediaAsset.file_type` |
| גודל | `MediaAsset.file_size` |
| תגיות | `MediaAsset.tags[]` |
| נושא | *`category/topic` (להוסיף) |
| מימדים / thumbnail / alt | *להוסיף |
| היכן בשימוש | *`usage[]` / `usage_count` (להוסיף — מעקב הפניות) |
| אחסון | Cloudflare R2 (קיים) + UploadFile/CreateFileSignedUrl |
