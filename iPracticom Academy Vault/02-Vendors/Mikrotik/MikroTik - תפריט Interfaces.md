# תפריט Interfaces (MikroTik)

ננווט בסרגל הניווט לתפריט **Interfaces**. זו החלונית הראשית שמציגה את כל ה"רגליים" של הראוטר — פיזיות ולוגיות, פנימיות וחיצוניות.

![[MikroTik-Interfaces-example.png]]
*(תמונה: מסך Interfaces עם עמודות Status ו-Traffic מסומנות)*

## עמודות מרכזיות
- **Name** — שם הממשק (למשל `vlan10-Phones`)
- **Type** — Ethernet / Bridge / VLAN / PPPoE וכו'
- **R / X** — Running (פעיל) / Disabled (מכובה)
