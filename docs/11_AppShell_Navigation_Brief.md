# בריף עיצוב — מעטפת הניווט (App Shell)

> **מטרה:** המסגרת הקבועה שכל המסכים יושבים בתוכה — סרגל צד + סרגל עליון.
> **מקורות:** PRD §11 (מפת עמודים) · SRS §1.1 (`profile_image_url`, `system_role`), SearchIndex.
> **בונה על:** כל המסכים 02–10. זו המעטפת שמאחדת אותם.

---

## 1. ההתאמה הקריטית — RTL

תמונת הרפרנס היא **LTR (סרגל בשמאל)**. iPracticom היא **עברית RTL** → הסרגל
עובר ל**צד ימין**, אזור התוכן בשמאל. זו ההתאמה המרכזית; אל תעתיק את ה-LTR.

> הלוגו בראש הסרגל (ימין-עליון ב-RTL) — מלא וקליק (→ דשבורד). כלל ה"לוגו 50%
> שקיפות" הוא לעמודי תוכן/שיעור, לא למעטפת.

## 3. סרגל הצד

**עליון:** לוגו (קליק → דשבורד) + כפתור כיווץ/הרחבה («).
**ניווט ראשי (תלוי-תפקיד):**

| פריט | יעד | תפקיד |
|---|---|---|
| דשבורד | `Dashboard` | כולם |
| ההכשרות שלי | `Learning` | כולם |
| פתרון בעיות | `Troubleshooting` | כולם |
| דשבורד ניהול | `ManagerDashboard` | manager+ |
| ניהול משתמשים | `UserManagement` | manager+ |
| ניהול תוכן | `ContentManager` | instructor+ |
| גיוס | `InvitationManagement` | manager+ |
| (אדמין: אשפים, אבטחה, מאגר AI) | — | admin |

**תחתון (קבוע — לפי בקשתך):**
- Help Center (מרכז עזרה)
- הגדרות (Settings)
- יציאה (Log Out)

**מצבים:** פריט פעיל מודגש · hover · סרגל מכווץ (אייקונים בלבד + tooltip).

---

## 4. סרגל עליון (Top bar)

- **כותרת העמוד** + תת-כותרת קצרה (משתנה לפי העמוד).
- **חיפוש גלובלי** — שדה חיפוש עם רמז ⌘K; פותח command palette שמחפש על פני
  שיעורים, מאמרים, מושגים ו-Playbooks (מעוגן ב-`SearchIndex`).
- **התראות** — אייקון פעמון + מונה לא-נקראו; לחיצה → פאנל התראות.
- **פרופיל** — תמונת פרופיל (`profile_image_url`) + שם; לחיצה → עמוד הפרופיל
  (תפריט מינימלי כדי לא לכפול את הגדרות/יציאה שבסרגל).

## 📋 הפרומפט — העתק מכאן

```
אני רוצה לעצב את מעטפת הניווט (App Shell) של iPracticom Academy — המסגרת
הקבועה שכל העמודים יושבים בתוכה. השתמש בשפת העיצוב והטוקנים הקיימים — RTL מלא.

— מותג: עברית RTL · #2EB4FF / #0075DB / #757D86 / #181D24 / רקע #F4FBFF ·
  tech-human · אייקוני outline · אנימציות עדינות.

— חשוב: RTL — הסרגל בצד ימין, אזור התוכן בשמאל (לא להעתיק LTR).

— סרגל צד (ימין) —
• עליון: לוגו iPracticom (קליק → דשבורד) + כפתור כיווץ/הרחבה.
• ניווט ראשי: דשבורד · ההכשרות שלי · פתרון בעיות · דשבורד ניהול ·
  ניהול משתמשים · ניהול תוכן · גיוס. (הצג הערה שהפריטים תלויי-תפקיד —
  משתמש רגיל רואה רק את 3 הראשונים.)
• תחתון (קבוע, מופרד בקו): מרכז עזרה · הגדרות · יציאה.
• פריט פעיל מודגש; מצב מכווץ = אייקונים בלבד + tooltip.

— סרגל עליון —
• כותרת העמוד הנוכחי + תת-כותרת קצרה (שמאל ב-RTL... כלומר בצד ההתחלה).
• שדה חיפוש גלובלי עם רמז ⌘K (command palette).
• אייקון התראות (פעמון) עם מונה לא-נקראו.
• פרופיל: תמונת פרופיל עגולה + שם המשתמש, קליק → עמוד הפרופיל.

— הצג —
1) המעטפת המלאה בדסקטופ עם עמוד דשבורד דמו בפנים, פריט "דשבורד" פעיל.
2) מצב הסרגל מכווץ (אייקונים בלבד).
3) ה-command palette של החיפוש פתוח עם תוצאות דמו (שיעור, מאמר, Playbook).
4) מובייל: הסרגל הופך לתפריט נשלף (drawer) + bottom bar או המבורגר.

— נתוני דמו: משתמש בעברית עם תמונת פרופיל, מחלקת תמיכה טכנית.
— טכני: React + Tailwind + shadcn/ui, lucide-react (outline), framer-motion.
  המעטפת היא layout שעוטף את תוכן העמודים (router outlet). שמור על רכיבים קיימים.

תתחיל מהמעטפת המלאה בדסקטופ עם הדשבורד בפנים.
```

## אנטומיה — רפרנס

| אלמנט | מקור |
|---|---|
| פריטי ניווט תלויי-תפקיד | `User.system_role` (user/instructor/manager/admin) → מפת עמודים PRD §11 |
| חיפוש גלובלי | `SearchIndex` (TF-IDF) → שיעורים/מאמרים/מושגים/Playbooks |
| תמונת פרופיל | `User.profile_image_url` |
| התראות | **לאמת/לבנות** — אין ישות התראות מפורשת ב-SRS |
