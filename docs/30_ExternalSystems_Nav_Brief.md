# בריף עיצוב — מערכות ארגוניות בסרגל הניווט

> **מטרה:** הוספת גישה למערכות חיצוניות בסרגל הניווט הגלובלי — לכל המשתמשים.
> **עדכון ל:** מעטפת הניווט (מסמך 11).

---

## 1. הסקשן

קבוצה חדשה בסרגל הצד — **"מערכות ארגוניות"** — מתחת לניווט הראשי, מעל הסקשן
התחתון (עזרה/הגדרות/יציאה). זמינה **לכל המשתמשים**. כל פריט נפתח ב**לשונית חדשה**
(קישור חיצוני, עם אייקון external-link עדין).

| מערכת | סוג | אייקון מוצע |
|---|---|---|
| Fireberry | CRM | אנשים/קשרי לקוחות |
| Priority (פריוריטי) | ERP | מבנה/תרשים |
| Mekhano (מכאנו) | שעון נוכחות | שעון |
| החדשנות | מרכזיות (PBX) | טלפון |
| מערכת נוספת | פנימית מותאמת | קובייה/אפליקציה |
| דרייב ארגוני | אחסון קבצים | תיקייה/דרייב |

---

## 📋 הפרומפט — העתק מכאן

```
עדכן את מעטפת הניווט של iPracticom Academy: הוסף סקשן "מערכות ארגוניות" לסרגל
הצד. אותה שפת עיצוב, RTL.

— מיקום: קבוצה נפרדת בסרגל (כותרת-קבוצה "מערכות ארגוניות"), מתחת לניווט הראשי
  ומעל הסקשן התחתון (עזרה/הגדרות/יציאה). מופרדת בקו עדין. זמינה לכל המשתמשים.

— פריטים (כל אחד נפתח בלשונית חדשה, עם אייקון external-link עדין בקצה) —
• Fireberry — CRM (אייקון קשרי-לקוחות)
• Priority — ERP (אייקון מבנה/ERP)
• Mekhano — שעון נוכחות (אייקון שעון)
• החדשנות — מרכזיות / PBX (אייקון טלפון)
• מערכת נוספת — אפליקציה פנימית (אייקון קובייה)
• דרייב ארגוני — אחסון (אייקון תיקייה)

— התנהגות: hover מציג tooltip עם שם מלא; הקבוצה ניתנת לכיווץ; במצב סרגל מכווץ —
  אייקונים בלבד + tooltip.

— הצג את הסרגל המלא עם הסקשן החדש משולב.
— טכני: React + Tailwind + shadcn, lucide-react (outline). שמור על רכיבים קיימים.

תתחיל מהסרגל עם הסקשן "מערכות ארגוניות" משולב.
```
