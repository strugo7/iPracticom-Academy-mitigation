# רישום שלוחה בטלפון — נוהל

אחרי שהשלוחה קיימת בשרת ([[הקמת שלוחה בשרת FusionPBX - נוהל]]), רושמים אותה על הטלפון הפיזי דרך **ממשק הניהול של הטלפון** (דפדפן, כתובת ה-IP של הטלפון). "רישום" (registration) הוא מה שהופך את השלוחה לקו עובד — ראו [[SIP - מושג כללי]]. הצעדים כאן הם עבור טלפון IP טיפוסי; שמות השדות עשויים להשתנות מעט בין דגמים.

**מה צריך מראש:** כתובת ה-IP של הטלפון (מתקבלת ב-DHCP), ומספר השלוחה + הסיסמה שנוצרו בשרת.

## כניסה לממשק הטלפון
שם משתמש וסיסמה בהגדרה ראשונית: `admin` / `admin`.

**דבר ראשון — משנים את הסיסמה** (לשונית **security**):
- שם משתמש: `admin`
- סיסמה: `ipXXXXpi` — כאשר `XXXX` הוא מספר הדומיין של הלקוח (לדוגמה `ip1842pi`).

## הגדרת השלוחה (לשונית Account)
1. בוחרים **Account** (`Account1` / `Account2` / `Account3` וכן הלאה).
2. **Line Active → Enable** — כדי שהקו (השלוחה) יהיה פעיל.
3. **Label** → מספר השלוחה.
4. **Display Name** → מספר השלוחה.
5. **Register Name** → מספר השלוחה.
6. **Username** → מספר השלוחה (**הכי חשוב** — זה מזהה הרישום מול המרכזייה).
7. **Password** → הסיסמה שלקחנו מהמרכזייה (זו שנוצרה ב[[הקמת שלוחה בשרת FusionPBX - נוהל]]).
8. **Server Host** → כתובת הדומיין המלאה מהמרכזייה.
9. **Transport → TCP**.
10. **Server Expires → `600`**.
11. **Codec** — מורידים את `G729` ואת `G722`.

> ⚠️ **פורט לפי מספר השרת:** פורט `5060` מיועד לשרתים `1,3,5,7`; פורט `5072` מיועד לשרתים `2,4`. מספר השרת נגזר מספרת הדומיין של הלקוח (ראו [[SIP - מושג כללי]] — תעבורה ופורטים).

![[phone-sip-account-registered-0c710fa7.png]]
*צילום מסך: לשונית Account בממשק הטלפון לאחר רישום מוצלח. שימו לב ל-**Register status = "Registered"** בראש; לשדות **Label / Display Name / Register Name / Username** שכולם מכילים את מספר השלוחה (בדוגמה — `315`); ולבלוק **SIP Server 1** שמכיל את כתובת הדומיין מהמרכזייה (`4001.ip-com.co.il`), **Port `5072`** (דומיין 4 → שרת זוגי), **Transport `TCP`** ו-**Server Expires `600`**. `SIP Server 2` נשאר ריק.*

## בדיקת הצלחה
מיד לאחר השמירה, ה-**Register status** של החשבון צריך להשתנות ל-**`Registered`**. אם הוא נשאר `Not Registered` / הטלפון מציג "אין שירות" — ראו [[אין שירות - טלפון לא נרשם]].
