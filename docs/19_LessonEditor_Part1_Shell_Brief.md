# בריף עיצוב — עורך השיעורים, חלק 1: המעטפת והפרדיגמה

> **מטרה:** היסוד של עורך השיעורים — פרדיגמת הבלוקים, הוספה, סידור, תצוגה מקדימה, גרסאות.
> **חלק 1 מתוך 4** (ראו מסמך 18 לאפיון המלא).
> **מקורות:** SRS §1.2 (ModuleLesson), §1.2.1 (בלוקי v2), LessonVersion.

---

## 0. החלטות נעולות

- **v2 בלבד** — עורך מבוסס-בלוקים. (v1 מומר לבלוקים; לא נתמך בעורך החדש.)
- **היסטוריית גרסאות** (LessonVersion) — דרישה מפורשת, מקבלת טיפול בולט.
- **פרדיגמה: עורך-בלוקים בסגנון מסמך, WYSIWYG, RTL** (Notion/Gutenberg) —
  הקנבס *הוא* השיעור.

---

## 1. Layout (RTL)

```
┌──────────────────────────────────────────────────────────────┐
│ סרגל עליון: [שם השיעור] [הגדרות] · [תצוגה מקדימה] [גרסאות]      │
│             [AI] · ⟳autosave · [פרסום]                        │
├──────────┬──────────────────────────────────┬────────────────┤
│ פלטת      │      קנבס — WYSIWYG               │  Inspector     │
│ בלוקים    │   הבלוקים כפי שהלומד יראה          │  (עיצוב/הגדרות  │
│ (5 משפ')  │   + "+" inline בין בלוקים          │  של בלוק נבחר)  │
│ (מתקפלת)  │   + סרגל הקשרי על בלוק נבחר        │  + Outline      │
└──────────┴──────────────────────────────────┴────────────────┘
```

- **פלטה** (ימין, מתקפלת): 6 משפחות הבלוקים לגילוי/גרירה.
- **קנבס** (מרכז): WYSIWYG — בלוקים נראים כמו ללומד.
- **Inspector + Outline** (שמאל): עיצוב/הגדרות הבלוק הנבחר + מבנה השיעור.

---

## 2. הוספת בלוק — שלוש דרכים משלימות

1. **"+" inline** — מופיע בין/אחרי בלוקים בריחוף; לחיצה → בורר בלוקים מקוטלג.
2. **פקודת-סלאש (`/`)** — בשורה ריקה, הקלדת `/` פותחת בורר מהיר (כמו Notion).
3. **פלטה צדדית** — 5 משפחות עם אייקונים, חיפוש, לגילוי וגרירה.

הבורר מקוטלג ל-5 המשפחות: טקסט/מבנה · מדיה · אינטראקטיבי · AI/הטמעה.

---

## 3. בלוק על הקנבס — Affordances

- בריחוף/בחירה: **ידית גרירה**, **+הוסף מעל/מתחת**, **סרגל הקשרי**
  (הזז, שכפל, מחק, עיצוב, הגדרות, נראות).
- מצב נבחר: טבעת הדגשה.
- **drag&drop** לסידור מחדש: ידית ברורה + אינדיקציית drop. + סידור במקלדת.
- העורך מציג כמה בלוקים פשוטים לדוגמה (cover, כותרת, טקסט, תמונה) רק כדי לקבע
  את הקנבס — עורכי הבלוקים המפורטים בחלקים 2-4.

---

## 4. היסטוריית גרסאות (LessonVersion) — בולט

פאנל/drawer "היסטוריית גרסאות":
- רשימת גרסאות: מספר גרסה, הערת שינוי (`description`), מי ערך (`created_by_name`),
  תאריך, ומונה בלוקים.
- לכל גרסה: **צפה** (תצוגה מקדימה לקריאה של ה-snapshot) + **שחזר**.
- שמירה ידנית יוצרת גרסה עם הערת שינוי; autosave שומר ברקע.

---

## 5. הגדרות השיעור (מהסרגל)

מודאל/פאנל: כותרת, מבוא (`introduction_text`), **מטרות למידה**
(`learning_objectives` — רשימה), משך (`duration_minutes`), XP (`xp_reward`),
אכיפת רצף (`require_previous_lesson`), מבחן מקושר (`linked_exam_id`), סטטוס.

— נתוני דמו: שיעור "מבוא ל-VLAN" עם בלוקים: lesson_cover, כותרת "מהו VLAN?",

— מצבים: דסקטופ-first. הצג גם את מתג התצוגה-מקדימה (עריכה ↔ תצוגת לומד).
