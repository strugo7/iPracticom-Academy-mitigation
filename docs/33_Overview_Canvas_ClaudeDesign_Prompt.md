## 📋 הפרומפט — העתק מכאן

```
אני רוצה שתבנה לוח Overview אחד (sitemap ויזואלי) שמציג את *כל* המסכים של
iPracticom Academy שעיצבנו בפרויקט הזה. השתמש במסכים הקיימים כ-frames/thumbnails
וסדר אותם על קנבס אחד גדול — אל תעצב מחדש, רק ארגן והצג.

— כותרת הלוח: "iPracticom Academy — מפת המערכת". RTL, שפת המותג
  (#2EB4FF / #0075DB / #757D86 / #181D24, רקע #F4FBFF, אייקוני outline).

— ארגן את המסכים באזורים מתויגים (כל אזור עם כותרת-קבוצה וצבע-מבטא) —

1) כניסה ומעטפת:
   • מסך התחברות
   • מעטפת הניווט (סרגל צד + top bar + מערכות ארגוניות)

2) מסע הלומד:
   • דשבורד אישי
   • ההכשרות שלי (קטלוג מסלולים)
   • תוכן ההכשרה (היררכיית מודול/נושא/שיעור)
   • נגן השיעור
   • מערכת ביצוע המבחן
   • פרופיל אישי

3) פתרון בעיות (Playbooks):
   • ספריית Playbooks
   • עורך Playbook
   • נגן Playbook — מובייל
   • נגן Playbook — דסקטופ

4) ניהול:
   • דשבורד ניהול

5) מפעל התוכן:
   • עץ התוכן
   • עורך השיעורים
   • מבחנים ומאגר שאלות
   • ספריית מדיה
   • מונחים (KMS)
   • יצירת תעודות
   • נהלים

6) הגדרות מערכת:
   • הגדרות מערכת
   • לוגי אבטחה
   • ניהול משתמשים ומבנה ארגוני

— מתחת לכל מסך: תווית עם שם המסך + תגית תפקיד (כולם / מנהל+ / מדריך+ / אדמין).

— חיבורים (זרימות): צייר חיצים עדינים שמראים את המסעות העיקריים:
   • התחברות → מעטפת → דשבורד.
   • דשבורד → ההכשרות → תוכן ההכשרה → נגן השיעור → מבחן → פרופיל (תעודה).
   • מעטפת → ספריית Playbooks → נגן.
   • מעטפת → מפעל התוכן (עץ התוכן → עורך השיעורים / מבחנים / מדיה / מונחים).
   • מעטפת → הגדרות (3 המסכים).

— מקרא צבעי-תפקיד בפינה.
— פריסה: קבוצות מסודרות במבט-על נקי, מרווחים אחידים, קריא בזום-אאוט.

תתחיל בפריסת האזורים והכותרות, ואז שבץ את המסכים הקיימים בכל אזור.
```

## טיפים

- אם Claude Design מתקשה עם הכל בבת אחת — בקש לבנות אזור-אזור ("עכשיו הוסף את
  אזור מפעל התוכן"), ואז לחבר בזרימות.
- אם חלק מהמסכים עדיין לא עוצבו בפרויקט (נהלים/תעודות/חדשים) — בקש placeholder
  עם שם המסך עד שתעצב אותם.
- לייצוא ל-HTML/שיתוף עם ה-PM: השתמש בפונקציית הייצוא/שיתוף של Claude Design על
  הלוח שנוצר.
