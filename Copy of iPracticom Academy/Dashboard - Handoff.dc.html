1 · סקירה

מסך Dashboard (נתיב ברירת המחדל, נגיש לכל התפקידים). מטרתו: שהלומד יראה במבט אחד היכן הוא עומד ויחזור בקליק ללמידה. כל הנתונים נגזרים מ-User.progress_stats (מחושב ע"י recalculateUserStats), מ-User.assigned_track_id, ומהפונקציות getLeaderboard ו-getActivityFeed. שני מצבים: מאוכלס (יש מסלול מוקצה) וריק (משתמש חדש ללא מסלול).

2 · עץ הרכיבים

<DashboardPage>                  // data fetching + state ('populated' | 'empty')
├─ <DashboardHeader>            // greeting · role tag · NotificationBell · logo
├─ if populated:
│   ├─ <ContinueLearningHero>   // assigned track + <ProgressRing> + CTA
│   ├─ <StatCardGrid>
│   │   └─ <StatCard> ×5        // XP · lessons · exams · certificates · time
│   ├─ <WeeklyActivityCard>     // <ScoreDonut> + weekly tiles + <WeeklyBars>
│   ├─ <Leaderboard>
│   │   └─ <LeaderboardRow> ×5  // current user highlighted
│   └─ <ActivityFeed>
│       └─ <ActivityItem> ×n
├─ else:
│   └─ <EmptyState>             // no assigned track → CTA + onboarding steps
└─ <DashboardFooter>

בנוי על Card מ-shadcn/ui · אייקונים lucide-react (outline, stroke 2) · אנימציות framer-motion (fade+rise עדין, 200–320ms).

5 · מצב ריק

תנאי: !user.assigned_track_id. במקום כרטיסים ריקים (XP=0, 0/0) מציגים <EmptyState> — קריאה לפעולה במקום "0" מתסכל:

  • כותרת "עוד לא שובצת למסלול למידה" + איור illus-support.png (spot-art, לא full-bleed).
  • שני CTA: בחירת מסלול (ראשי) → Learning · לספריית הידע (ghost) → Knowledge.
  • כרטיס "שלושה צעדים להתחלה": השלמת פרופיל · סרטון פתיחה · בחירת מסלול.
  • הכותרת והפוטר נשארים זהים. ה-KPI / מובילים / פיד מוסתרים לחלוטין (לא מוצגים כ-0).

6 · מובייל ורספונסיביות

  • Hero נשאר בראש; מתחת md עובר ל-flex-col והטבעת ממורכזת מתחת לטקסט.
  • KPI: grid-cols-2 במובייל, md:grid-cols-3, xl:grid-cols-5 (או auto-fit minmax(196px,1fr)).
  • אזור ראשי (מובילים/פעילות): מ-lg:grid-cols-[1.55fr_1fr] נערם ל-grid-cols-1 מתחת ל-lg — הפיד יורד מתחת.
  • כותרת: החיפוש/פרטים מתקפלים; שומרים ברכה + פעמון + לוגו. hit-target ≥ 44px.
  • RTL מלא: dir="rtl", ps-/pe- במקום left/right, flex-row מתהפך אוטומטית.
