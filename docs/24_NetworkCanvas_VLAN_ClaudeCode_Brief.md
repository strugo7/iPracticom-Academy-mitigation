# בריף ל-Claude Code — סימון VLAN בטופולוגיית הרשת

> **מטרה:** הוספת יכולת קיבוץ רכיבים ל-VLANs (פוליגון/קבוצה צבעונית) לבלוק
> `network_canvas` שכבר נבנה.
> **קהל:** Claude Code (תוספת לרכיב קיים, לא עיצוב חדש).

---

## 1. הקשר

הבלוק `network_canvas` כבר קיים (@xyflow/react): רכיבי רשת כצמתים + חיבורים +
פאנל סוויץ'. **התוספת:** לעטוף קבוצות רכיבים בפוליגון צבעוני שמייצג VLAN.

---

## 2. מבנה הדאטה (תוספת)

מוסיפים מערך ל-`data` של הבלוק — בלי לשנות קיים:

```js
vlans: [
  { id, name, vlan_id, color, member_node_ids: [nodeId, ...] }
]
```

- כל VLAN מצביע על הצמתים שבו (`member_node_ids`).
- צומת יכול להשתייך ליותר מ-VLAN אחד (trunk) — לטפל בחפיפה ויזואלית.
- **תאימות-לאחור:** ברירת מחדל `vlans: []`. טופולוגיה קיימת (מיובאת מ-Base44)
  פשוט לא תכיל VLANs — לא נשבר כלום.

---

## 3. הגישה הטכנית

**מומלץ — להתחיל מ-React Flow group nodes** (מובנה): צומת-קבוצה שרכיבים יושבים
בתוכו; גבול אוטומטי, הזזת קבוצה מזיזה הכל. נותן 90% מהערך בפשטות.

**שדרוג עתידי — פוליגון SVG חופשי:** `<polygon>`/`<path>` על שכבת background
(zIndex מתחת לצמתים), עם convex hull סביב חברי ה-VLAN. רק אם צריך צורות אורגניות.

> **קריטי לפני מימוש:** בדוק באיזו גרסת React Flow הבלוק הקיים השתמש ואיך נבנתה
> שכבת הרקע/הצמתים — כדי להשתלב ולא לסתור (במיוחד אם כבר יש group nodes או zIndex).

---

## 4. זרימת המשתמש

1. בחירת רכיבים (Shift+click או גרירת-בחירה).
2. לחיצה "קבץ ל-VLAN" → פאנל: שם, VLAN ID, צבע.
3. נוצר פוליגון/קבוצה צבעוני-שקוף שעוטף את הרכיבים, עם תווית VLAN בפינה.
4. כל VLAN בצבע שונה; חפיפה (trunk) נראית ויזואלית.
5. עריכה/פירוק VLAN מהפאנל.

---

## 📋 פרומפט ל-Claude Code — העתק מכאן

```
הוסף יכולת VLAN לבלוק network_canvas הקיים (@xyflow/react). שלבים:

1. לפני הכל: בדוק את המימוש הקיים — גרסת React Flow, איך הצמתים והרקע מסודרים
   (zIndex), והאם כבר נעשה שימוש ב-group nodes. השתלב עם הקיים.

2. מבנה דאטה: הוסף ל-data של הבלוק מערך:
   vlans: [{ id, name, vlan_id, color, member_node_ids: [] }]
   ברירת מחדל [] — אסור לשבור טופולוגיות קיימות בלי השדה.

3. רינדור: עטוף את חברי כל VLAN בפוליגון/קבוצה צבעוני-שקוף, מתחת לצמתים
   (zIndex נמוך), עם תווית VLAN (שם + VLAN ID) בפינה. השתמש ב-group nodes
   של React Flow כברירת מחדל. טפל בצומת שבכמה VLANs (trunk) — חפיפה ויזואלית.

4. אינטראקציה: בחירת רכיבים (Shift+click / גרירת-בחירה) → כפתור "קבץ ל-VLAN"
   → פאנל (שם, VLAN ID, בורר צבע) → יצירת ה-VLAN. אפשר עריכה ופירוק.

5. שמירה: עדכן את data.vlans בבלוק; ודא שזה נשמר ב-blocks[] של השיעור
   ובהיסטוריית הגרסאות.

שמור על RTL, שפת העיצוב הקיימת, והנגישות. אל תשבור את פאנל הסוויץ' או החיבורים
הקיימים.
```

---

## 5. הערות

- **מיגרציה:** `vlans` הוא שדה תוסף עם ברירת מחדל ריקה — נכנס נקי, לא נוגע בקיים.
- **trunk:** צומת בכמה VLANs — ודא שהפוליגונים שקופים-חלקית כך שחפיפה נראית.
- **עקביות צבעים:** הקצה צבעים מובחנים אוטומטית, או תן למשתמש לבחור מהפלטה.
